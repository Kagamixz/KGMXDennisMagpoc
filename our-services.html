  • Central Processing Unit (CPU)

    Often referred to as the brain of the computer, the CPU performs calculations and executes instructions. It's crucial for overall system performance.

  • Random Access Memory (RAM)

    RAM is volatile memory that provides temporary storage for data that the CPU is currently using or processing. It affects system speed and multitasking capabilities.

  • Motherboard

    The motherboard is the main circuit board that houses the CPU, RAM, and other essential components. It facilitates communication between different parts of the computer.

  • Storage (Hard Drive or SSD)

    Storage devices, like hard drives (HDD) and solid-state drives (SSD), store the operating system, applications, and user data. SSDs are faster but typically more expensive than HDDs.

  • Graphics Processing Unit (GPU)

    Also known as a graphics card or video card, the GPU is responsible for rendering images and videos. It's crucial for gaming, graphic design, and video editing.

  • Power Supply Unit (PSU)

    The PSU provides electrical power to the components in the computer. It converts electrical power from an outlet into a form that the computer can use.

  • Case (Chassis)

    The case holds and protects the internal components of the computer. It also provides a platform for mounting the motherboard, power supply, and other hardware.
